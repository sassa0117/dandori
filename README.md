# だんどり — フロー → AI

作業の流れ（ワークフロー）を視覚的に組み立てて、**AIに渡す手順テキスト（Markdown / Mermaid図）に変換**するツール。
テキストだけで指示すると接続詞で齟齬が出る問題を、図で組んでから渡すことで解消します。

公開URL: https://sassa0117.github.io/dandori/

## 使い方
1. どこでも**ダブルクリック**で手順を1つ追加（そのまま入力）
2. 箱の**下の○を次の箱へドラッグ**して矢印でつなぐ
3. 種類（開始／分岐／終了）は**つなぎ方から自動判定**。自分で選ばない
4. 「📋 AIに渡す」でテキスト化 → コピーしてClaude等に貼る

- データはブラウザに自動保存（localStorage）。サーバー不要
- 単一HTML。`index.html` を直接開いても全機能動く
- スマホはブラウザで開いて「ホーム画面に追加」でアプリ化（オフライン対応）
